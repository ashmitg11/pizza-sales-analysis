# Pizza Sales SQL Analysis

## Project Overview
This project analyzes pizza sales data using SQL to uncover business insights related to revenue trends, customer behavior, and product performance.

## Tools Used
- SQL
- MySQL
- Excel

## Key Business Questions
- What are the top-selling pizzas?
- Which pizza category generates highest revenue?
- What are the peak ordering hours?
- Which pizza size is most preferred?
